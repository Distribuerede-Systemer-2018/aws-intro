## Setup AWS server

Start serveren hos amazon, husk at downloade key pair

Åben terminalen og kør 
```chmod 0400 [pem fil]```
```ssh -i [pem fil] ec2-user@IP```

Gode kommandoer:

Installer et program via package manager (fx java)
```sudo yum install [program]``` 

Download en fil
```wget [fil]```

Udpak en tar fil
```tar xvf [fil]```
